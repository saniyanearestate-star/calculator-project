# 🧮 Python Calculator (CLI)

A simple and efficient command-line calculator built using Python.  
This project demonstrates core programming concepts like functions, conditionals, loops, and user input handling.

---

## 📌 Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division (with zero-division handling)
- 🔁 Continuous calculations using loop
- 🧠 Clean and modular code using functions

---

## 🛠️ Tech Stack

- Python 3
